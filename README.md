# File Encryption System

## Description
This project is a File Encryption System using symmetric cryptography.
It allows users to securely encrypt and decrypt files using a secret key.

## Features
- Encrypt files securely
- Decrypt files using the correct key
- Protect sensitive information
- Simple user interface

## Technologies Used
- Python
- Flask
- HTML/CSS
- Cryptography library

## Project Structure

File-Encryption-System/
│
├── app.py
├── templates/
│   └── index.html
└── uploads/

## Objective
The goal of this project is to implement symmetric cryptography
for secure file protection and data confidentiality.
